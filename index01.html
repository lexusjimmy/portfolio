skyfie

2015-2017
  • Genre: Research
  • Responsible for: Design, Develop
  • Technique: Unity, Swift, Arduino

As personal drones become more popular, we can envision a future where flying selfie bots are always with us. At present, users have to be very skillful to pilot the drone and are not easy to take aerial selfies by state-of-the-art methods. Based on user observation, we generalize the interaction flow of selfie taking into two stages: Positioning and Framing stage. In each stage, we present new interaction techniques including a direct pointing technique, two fine-tuning techniques, and a touch manipulation for framing. We conducted an experiment in VR and found that positioning a camera by using our Cylindrical and Spherical techniques are more efficient and easier to learn and use. We implemented a proof-of-concept Skyfie controller for evaluation. The result shows users felt intuitive and expressed enthusiasm to take aerial selfies with our techniques. Finally, we discuss the insights from the evaluation and conclude with future directions.

EasyAuction

2016
  • Genre: Website
  • Responsible for: Design, Programming
  • Technique: html, CSS, JavaScipt

EasyAuction is a website that allows people to upload their items for sold. Visitors can log-in with facebook account and leave messages to sellers. A basic and easy website for second-hand goods selling.
Since this website is powered by Google firebase, it updates the messages immediately. You can even regard it as a messenger.
Although it has database to save the data that users upload, I didn't use any of the backend technology. Simply using the Jquery and JavaScript accomplish it.

BBR

2014
  • Genre: Tools
  • Responsible for: Programming(Arduino), Concept
  • Technique: Arduino, html, JavaScript

BBR, which is standed for "Better Badminton Revolution" is an original training tool for badminton. We found out the problem of lacking badminton practicing place, while this sports require highly technique.
We invented this system to help people practice the steps of badminton. Inspired by the video game "Dance Dance Revolution", we designing our tool to let user practice while playing our tool. User can use it without going to badminton field.
With these innovation, this tool won "金驗獎".
This system consist three hypersonic distance detector and one android phone. Using the node- js we can read the result from detector and conduct the game.
